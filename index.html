Core Web Technologies

HTML (Hypertext Markup Language) is the backbone of web content, providing structure and semantics for web pages.

CSS (Cascading Style Sheets) is used for styling and layout of web pages, controlling the visual presentation of HTML elements.

JavaScript is a programming language that enables interactive web pages, allowing for dynamic content updates and user interaction.

Additional Web Technologies

React

A popular JavaScript library for building user interfaces.

Node.js

A runtime environment that allows JavaScript on the server-side.

SQL

A language for managing and querying relational databases.
